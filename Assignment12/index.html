Best Music Bot

I like to make Discord bots in my free time. This bot is nearly done. It comes with a web app hosted on Firebase that stores and manages a music queue. You can search for song via the YouTube API to find songs so that you don't need to paste a link.

To get the bot to work (in theory because it's not done), you need to long into the web app, click on the invite link, add the bot to your Discord server, and then you can start using the bot to listen to music.
